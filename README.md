📝 To-Do List Application (Python)

📌 Project Description

The To-Do List Application is a simple command-line Python program that helps users organize and manage their daily tasks. It allows users to add new tasks, view all saved tasks, and exit the application through an easy-to-use menu. This project is ideal for beginners to practice Python programming concepts.

✨ Features

Add new tasks.

View all saved tasks.

Interactive menu-driven interface.

Handles empty task lists gracefully.

Simple and user-friendly.


🛠️ Technologies Used

Python 3


📂 How It Works

1. The program displays a menu with available options.


2. Users can choose to:

Add a new task.

View all tasks.

Exit the application.



3. Tasks are stored in a Python list during program execution.


4. The application continues running until the user chooses to exit.
